> **vue**：一个mvvm框架（库），和 angular 类似，比较容易上手、小巧。

官网：https://cn.vuejs.org/
手册：https://cn.vuejs.org/v2/api/
```
vue 和 angular 区别？

vue ―― 简单易学
       指令以 v-xxx
       一片html代码配合上json，再new出来vue实例
       个人维护项目
       适合：移动端项目
angular ―― 上手难
       指令以 ng-xxx
       所有属性和方法都挂到$scope身上
       由google维护
       适合：pc端项目

共同点：不兼容低版本IE
```

vue雏形：
```
<div id="app">
    {{msg}}
</div>

var vm = new Vue({
    el: '#app',  // 选择器：id、class、tagName
    data:{
        msg:'hello vue'
    }

})
```
